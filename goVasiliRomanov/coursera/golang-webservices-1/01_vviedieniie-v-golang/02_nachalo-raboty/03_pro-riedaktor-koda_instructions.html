В курсе используется бесплатный редактор Visual Studio Code ( https://code.visualstudio.com/ )со светлой темой оформления. Причина выбора данного редактора - большой красивый терминал, открываемый поверх кода.

Ещё альтернативы:

  • Sublime Text 3 - https://www.sublimetext.com/ + плагин для golang

  • Vim-go - https://github.com/fatih/vim-go

  • GoLand - https://www.jetbrains.com/go/ - платный редактор от JetBrains. Хорошо подойдёт тем, кто активно пользуется их софтом

  • Lite IDE - https://github.com/visualfc/liteide

----

Для пользователей Windows:

Power Shell в некоторых местах эскейпит больше чем надо и рекомендуется запускать код и тесты через bash из комплекта Git For Windows.

Для того чтобы подключить git bash в Visual Studio Code надо прописать в настройках https://s.mail.ru/6WPp/UcymwHcK7 

"terminal.integrated.shell.windows": "C:\\Program Files\\Git\\bin\\bash.exe",
